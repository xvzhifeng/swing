# v2.0.0-beta 发布指南

## 📦 发布信息

- **版本号**: v2.0.0-beta
- **发布日期**: 2026-06-15
- **Git 标签**: `v2.0.0-beta`
- **状态**: Beta（待真机测试）

## 🎯 Beta 版本包含内容

### 核心功能 (Phase 2)
✅ 击球类型分类
  - 杀球 (Smash) - 向下击球
  - 抽球 (Drive) - 水平击球  
  - 挑球 (Lift) - 轻柔向上击球
  - 高远球 (Clear) - 强力向上击球

✅ 实时数据显示
  - 运动中显示4种击球类型计数
  - 当前速度大卡片
  - 完成界面详细统计

### 额外功能
✅ 自定义校准模式
  - 标准模式: 2.0g
  - 专业模式: 2.5g
  - 自定义模式: 用户自主校准
  - 5次挥拍平均值×0.8算法
  - 阈值持久化存储
